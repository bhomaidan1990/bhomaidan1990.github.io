Research Fields

Research domains that I'm Interested in.

Computer Vision

My interests are camera calibration, object tracking, and automatic object detection. In addition to classical image processing techniques using OpenCV and Scikit-Image.

Machine Learning

My interests here are Image manipulation (classification, segmentation, etc...), Natural Language Processing (NLP), and Deep Learning using Tensorflow, Keras, and PyTorch. In addition to machine learning algorithms using Scikit-learn and Pandas.

Robotics and Artificial Intelligence

I'm trained on ABB IRB 14000 YuMi, using low-level RAPID language to move the robot, and giving it the power to think using AI techniques, in particular I'm interested in Automated Planning(AP).
